## 3. Accessibility considerations
- What accessibility features did you include?
  Accessibility features included:

Added a skip link at the top of the page (<a href="#main">Skip to content</a>).

Added alt text for all images to describe their content.

Used semantic HTML elements like <header>, <section>, <main>, <figure>, and <footer>.

Followed a proper heading hierarchy (h1 → h2 → h3).
- How do they improve usability?
- The skip link helps keyboard and screen-reader users jump directly to the main content without navigating through the whole page.

Alt text allows visually impaired users using screen readers to understand what the images represent.

Semantic elements help assistive technologies understand the structure of the page.

Correct heading order makes navigation easier for screen-reader users and improves overall readability.
